# 💪 My Coding Challenges (Typescript)

A repository filled with my coding challenges using typescript.

Clone this repository:

```
git clone https://github.com/ijborda/coding-challenges-typescript.git
```

Install the dependencies:

```
npm install
```

Test a file with the following command (Each file is one kata/problem):

```
npm run test {file.ts}
```

Example:

```
npm run test codewars/8kyu_001.ts
```

## Requirement
- Node 14

## Debug steps
Note: These steps need to be performed again on every file change.
1. Execute `npm run build`.
2. Open the file to debug.
3. Comment out the test part of the file and include a function execution line. Be sure to also place the breakpoint(s).
4. Run the vscode debug with `Run Current File`.
